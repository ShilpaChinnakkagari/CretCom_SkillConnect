# SkillConnect - Local Services Marketplace

A full-stack platform connecting customers with local service providers.

## Tech Stack
- **Frontend:** React.js + Tailwind CSS
- **Backend:** Spring Boot
- **Database:** MongoDB Atlas
- **Auth:** JWT + Google OAuth

## Project Structure
- `/backend` - Spring Boot API
- `/frontend` - React Web App
- `/mobile` - Flutter App (Coming Soon)
- `/docs` - Documentation
